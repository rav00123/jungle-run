# Jungle Run

A 2D endless runner game developed using Unity and C#.

## Overview

Jungle Run is an endless runner game where the player navigates through obstacles, collects points, and attempts to achieve the highest possible score.

## Features

* Dynamic obstacle spawning
* Score tracking system
* Scene management
* Prefab-based architecture
* Game over and restart functionality
* Collision detection system

## Technologies Used

* Unity
* C#
* Visual Studio

## Responsibilities

* Developed gameplay mechanics and player controls
* Implemented obstacle spawning and scoring systems
* Designed UI elements and game state management
* Utilized Unity prefabs for reusable components
* Tested and debugged gameplay features

## How to Run

1. Clone the repository
2. Open the project in Unity Hub
3. Open the main scene
4. Press Play
